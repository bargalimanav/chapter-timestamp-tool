🔑 Google API Configuration
How to get these keys (one-time setup):
1. Go to Google Cloud Console → Create or select a project.
2. Enable the Google Sheets API from the API Library.
3. Go to APIs & Services → Credentials.
4. Create an API Key → paste it above. Restrict it to Sheets API for security.
5. Create an OAuth 2.0 Client ID (type: Web application) → add https://your-username.github.io as an Authorized JavaScript Origin → paste the Client ID above.
6. Keys are saved in your browser's localStorage — they never leave your device.
Make sure the sheet is shared with your Google account.
